算法-左孩子右兄弟

【算法复习】左孩子右兄弟

​ 其实还是用DP的思想找规律，我们假设某个节点共有K个子节点，这K个子节点的最高高度分别为p1,p2,p3······pk，那么总树的最高高度为H = K + pk。

​ 上式相当于将最高的子树放到最后，很好理解，如果最高的子树放在中间则会有部分长度被浪费（兄弟链的长度是确定的K）。

​ 用vector来表示邻接表存储树，很方便的就能实现：

#include "stdio.h"
#include "vector"
#include "algorithm"
using namespace std;
vector<int> leaves[100005];//邻接表存储树
int f[100005] = { 0, };//存储子树高度，加快dp时间
int H(int t) {
    if (leaves[t].size() == 0) return 0;//叶子结点返回0高度
    else if (f[t] == 0){//非叶子结点
        int m = 0;
        for (int i = 0; i < leaves[t].size(); ++i) m = max(m, H(leaves[t][i]));
        f[t] = leaves[t].size() + m;//兄弟链长度+子树最大长度
    }
    return f[t];
}
int main(void){
    int N,t;
    scanf("%d", &N);
    for (int i = 2; i <= N; ++i) {
        scanf("%d", &t);
        leaves[t].push_back(i);
    }
    printf("%d\n", H(1));
    return 0;
}
