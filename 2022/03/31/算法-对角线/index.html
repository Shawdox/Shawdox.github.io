算法-对角线

【算法复习】对角线

​ 用DP递推不好做，逻辑推导题，任何三条对角线都不会交于一点，故一个交点有且仅有两条直线，唯一确定四个不同的顶点，故交点个数为C^4^ ~n~.

​ 故ans = n (n-1) (n-2) *(n-3)/24，直接这么做long long 会爆：

#include "bits/stdc++.h"
int main(void) {
	unsigned long long ans = 0,n;
	scanf("%llu", &n);
	ans = n * (n - 1) / 2 * (n - 2) / 3 * (n - 3) / 4;
	printf("%llu\n", ans);
	return 0;
}//end of main
