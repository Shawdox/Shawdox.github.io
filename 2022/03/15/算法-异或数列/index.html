算法-异或数列

【算法复习】异或数列

​ 博弈论问题，首先，数列中的每个数可以分成每一位来看待，某一位数，对应的数列应为一串0和1组成。在这一位上的博弈情况有：

1. 自己有奇数个1，对面有偶数个1，自己胜出；
 2. 反过来对面奇数个1，自己偶数个1，对面胜出；
 3. 平局时双方1的个数相同。

​ 由以上性质可以看出，博弈的关键是对1的抢占。

  • 当数列中1的个数为偶数时，无论怎么划分要么两边分到的1都是偶数，要么两边分到的1都是奇数，故一定是平局。
  • 当数列中的1的个数为奇数时，同理，不存在平局的情况，如果只有一个1，则先手必赢，若1的个数大于1：
    • 若0的个数为奇数：若先手方一手抢‘1’，则还剩偶数个‘1’，后手方可以抢‘0’，来保证最后一个’1’一定在他手里，后手方胜利；
    • 若0的个数为偶数：同理，先手方手里。
#include "stdio.h"
using namespace std;
int main(void){
    int T, n, c[21],ans;//c[i]表示第i位1的个数
    long t;
    scanf("%d", &T);
    for (int i = 0; i < T; ++i) {
        scanf("%d", &n);
        memset(c, 0, sizeof(c));
        ans = 0;
        for (int j = 0; j < n; ++j) {
            scanf("%ld", &t);
            for (int k = 1; k <= 20; ++k) c[k] += (t >> (k - 1)) & 1;
        }
        for (int k = 20; k > 0; --k) {
            if (c[k] % 2 != 0) {
                if (n % 2 == 0 && c[k]!= 1) ans = -1; 
                else ans = 1;

                break;
            }
        }
        printf("%d\n", ans);
    }
    return 0;
}
