CSP复习-直线交点

【CPS复习】直线交点

​ 大体思路是将图案中所有的平行线分为一组，另外的直线分为另一组，则两组线有r(N-r)个交点，因为平行线组内没有内部交点，故在有r条平行线的情况下，有r (N-r) + [N^’^ == N-r时的方案]个交点。

​ 由于对某一N，其情况总和为从0条平行线到N条平行线，每一种子情况又分为N-r个方案，故不好用一个递归函数g(x)直接表示N=x时的种数，令函数g(n,k)代表一个处理过程，n表示待计算的线的条数，k代表以已经计算出的交点个数：

#include "stdio.h"
bool f[1000000] = { 0, };
int ans = 0;
void g(int n, int k) {
    if (n == 0) {
        if (f[k] == false) ans++;
        f[k] = true;
    }
    else for (int i = 1; i <= n; ++i) g(n - i, i * (n - i) + k);
}
int main(void) {
    int N;
    scanf("%d", &N);
    g(N, 0);
    printf("%d\n", ans);
    return 0;
}
