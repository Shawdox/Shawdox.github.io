CSP复习-拍头

【CPS复习】拍头

​ 直接暴力的时间复杂度是O(N^2^) = 1 × 10^10^，肯定超时。

​ 观察题目可以发现，某个奶牛需要拍头的数量=其A_i的所有约数A_j的数量和:

​ 例如A_i = 12 = 2 2 3，则奶牛i需要拍头的数量为:num[1]+num[2]+num[3]+num[4]+num[6]+num[12]-1，减一是为了防止自己拍自己的头。

​ 故如果能够在输入的时候用数组num[]统计每个值的数量，然后对每个A_i分解其约数再对其数量相加即可，因为一个数x的约数 < $2\sqrt{x}$，故总算法的时间复杂度为$O(N\sqrt{x})= 10^8$.

​ 具体实现时，可以反过来想，每个数A_i对其倍数都贡献num[A_i]（对自己要-1）：

#include "stdio.h"
#include "algorithm"
using namespace std;
int A[100005] = { 0, };
int num[1000005] = { 0, };//计数
int ans[1000005] = { 0, };//最终统计结果
int main(void){
    int N,Max = 0;
    scanf("%d", &N);
    for (int i = 1; i <= N; ++i) {
        scanf("%d", &A[i]);
        ++num[A[i]];
        Max = max(Max, A[i]);
    }
    for (int i = 1; i <= Max; ++i) {//注意这里不能遍历A_i，否则会重复计算
        if (num[i] != 0) {
            for (int j = 1; j * i <= Max; ++j) {
                ans[j * i] += num[i];
            }
        } 
    }
    for (int i = 1; i <= N; ++i) printf("%d\n", ans[A[i]] - 1);//减一防止
    return 0;
}
